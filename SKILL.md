# samectx

Context management tool - 智能整理对话上下文并保存到本地目录

## 核心功能
智能分析对话内容,提取关键任务、关键点和关键决策,生成 `samectx sync` 命令并自动执行。

## 触发条件
- 用户提到「保存上下文」、「同步笔记」、「记录对话」、「整理对话」、「Sync Context」、「Save Context」等关键词
- 用户直接请求保存当前对话

## 工作流程
1. **分析对话**: 识别对话中的关键任务、关键点和关键决策
2. **生成命令**: 构建带参数的 `samectx sync` 命令
3. **自动执行**: 使用 RunCommand 工具自动执行命令

## 指令
当用户请求同步上下文时，请按以下步骤操作:

### 步骤 1: 分析对话内容
识别对话中的:
- **关键任务**: 待办事项、开发任务、计划
- **关键点**: 技术要点、重要发现、注意事项
- **关键决策**: 技术选型、架构决策、方案选择

### 步骤 2: 提取关键信息
从对话中提取:
- 关键任务列表
- 关键点列表
- 关键决策列表

### 步骤 3: 生成命令
构建带参数的 `samectx sync` 命令:
```bash
samectx sync --tasks "任务1;任务2" --keypoints "关键点1;关键点2" --decisions "决策1"
```

### 步骤 4: 自动执行
使用 RunCommand 工具自动执行生成的命令。

## 示例
**用户**: "我需要实现用户登录功能,使用 JWT 进行认证,选择 bcrypt 加密密码"

**技能响应**:
1. 分析对话内容:
   - 关键任务: 实现用户登录功能
   - 关键点: 使用 JWT 进行认证
   - 关键决策: 选择 bcrypt 加密密码
2. 生成命令:
```bash
samectx sync --tasks "实现用户登录功能" --keypoints "使用JWT进行认证" --decisions "选择bcrypt加密密码"
```
3. 自动执行命令

## 注意事项
- 笔记将保存在项目目录的 `samectx-notes/{username}/` 文件夹中
- 建议将笔记目录提交到 Git 仓库以支持跨设备同步
- 请勿在笔记中包含敏感信息(密码、密钥等)
